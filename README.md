# Countries
Simple application that shows you countries and details of a specific country which you choose

## Technologies
- Hilt (Dependency Injection)
- MVVM (Architecture)
- Coroutines (Asnchronous Operations)
- Room
- Retrofit (Network Operations)
- Mockito (Unit Tests)
- ViewBinding
- Navigation Component
- Safe-Args
- GlideToVectorYou
- Swipe Decorator
- Lottie Animation
